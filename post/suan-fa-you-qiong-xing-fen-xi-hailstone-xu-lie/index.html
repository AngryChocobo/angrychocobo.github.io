算法有穷型分析 hailstone序列

2019-08-07

学到了一个知识点：

hailstone序列：
1）输入一个正整数n；
（2）如果n=1则结束；
（3）如果n是奇数，则n变为3n+1，否则n变为n/2；
（4）转入第（2）步。

function hailstone(n) {
	var length = 1; 
	while(1 < n) {
		if(n % 2 === 0) {
			n = n/2;
		} else {
			n = 3 * n + 1;
		}
		length++;
		
	}
	return length;
}
