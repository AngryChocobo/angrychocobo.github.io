node聊天室

2019-09-19

跟着《node.js实战》实现一个聊天室

需求：

  • 消息发送后会展示给房间内的所有用户
  • 进入房间，系统随机分配一个昵称，可以使用命令的形式修改昵称
  • 用户可以使用命令的形式创建聊天室（或加入）
  • 聊天室名显示为网页的title
  • 消息区域右侧有可用房间列表

任务分解

  1. 样式编写
  2. 服务端处理聊天信息
  3. 浏览器处理聊天信息

需要用到的库或模块

  • socket.io实现WebSocket
  • node-mime

建立项目及准备工作

项目建立就不多说了，项目中建立文件夹public，用来保存静态文件，并在根目录创建server.js

使用node实现一个httpServer

const http = require('http');
const fs = require('fs');
const path = require('path');
const mime = require('mime');

let cache = {};

function send404(res) {
    res.writeHead(404, {
        'Content-Type': 'text/plain;charset=utf-8'
    });
    res.write('Error 404: 资源未找到～');
    res.end();
}

function sendFile(res, filePath, fileContent) {
    res.writeHead(200, {
        'Content-Type': mime.lookup(path.basename(filePath))
    });
    res.end(fileContent);
}

function serveStatic(res, cache, absPath) {
    if (cache[absPath]) {
        sendFile(res, absPath, cache[absPath]);
    } else {
        fs.exists(absPath, exists => {
            if (exists) {
                fs.readFile(absPath, (err, data) => {
                    if (err) {
                        send404(res);
                    } else {
                        cache[absPath] = data;
                        sendFile(res, absPath, data);
                    }
                });
            } else {
                send404(res);
            }
        });
    }
}

const server = http.createServer((req, res) => {
    console.log(req.url);
    let filePath = false;
    if (req.url === '/') {
        filePath = 'public/index.html'
    } else {
        filePath = 'public' + req.url;
    }
    const absPath = './' + filePath;
    serveStatic(res, cache, absPath);
});

server.listen(5555, () => {
    console.log('server listening on port 5555');
})

这里要注意：Content-Type需要加上charset=utf-8来保证中文不会乱码

中间犯的错误

  • absPath错误的写成了:
const absPath = '/ + filePath;
  • server 运行中，修改静态文件不能看到效果，必须重启server
