Vue实现视频网站

2019-11-06

首先开始做管理端，顺便整理思路
技术栈：nest.js
安装nest.js npm install -g @nestjs/cli
创建管理端： nest new server
创建单体项目： nest g app admin

至此，后台存在两个项目admin和server。

启动admin项目：nest start -w admin

创建admin和server公共的库(数据库模块)： nest g lib db

安装nest的数据库包：yarn add nestjs-typegoose @typegoose/typegoose
安装moogoose： yarn add mongoose @types/mongoose

在libs/db/src/db.module.ts中引入typegoose模块
