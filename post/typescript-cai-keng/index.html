放到vue项目中后，提示import vue页面报错

在项目目录创建x.d.ts文件：

declare module '*.vue' {
  import Vue from 'vue'
  export default Vue
}

之后在ts的配置文件种引入:

"include": [
    // 需要编译的ts文件一个*表示文件匹配**表示忽略文件的深度问题
    "./src/**/*.ts"
  ],

最后如果发现错误还存在的话，重启一下vscode，就会发现报错消失了（坑我半小时）
燃鹅，此时vscode点击vue文件的路径跳转的话，会发现打开的不是我们声明的vue文件，而是声明的类型文件
